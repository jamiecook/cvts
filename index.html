Glossary¶

This glossary is tuned to how we use the terms within CVTS, which is often different on how they are used more generally.

geometry¶

A spatial object (at present, a Polygon or MultiPolygon) that designates some area of interest.

geography¶

A set of geometries that specify regionalisations of interest. see here for a description of what is required in this package.

stop point¶

A location where a vehicle has not moved for more than cvts.settings.MIN_STOP_TIME seconds.

trip¶

A journey between two stop points.
